# Health Grid — Portal del Paciente

## Modulo 2 - Turnos

Aplicación web construida con **React**, **TypeScript**, **Vite**, **Tailwind CSS** y **Shadcn ui**

## Requisitos

- **Node.js 22**
- **pnpm**

## Instalación

En la raíz del proyecto:

```bash
pnpm install
```

## Iniciar en desarrollo

```bash
pnpm run local-server   '>> Inicia servidor local simulando endpoints'
pnpm run dev            '>> Inicia el proyecto'
```
